🌙 Eid Card Generator – Python + Streamlit

Create beautiful, shareable Eid cards with flying stickers, animated moon, and heartfelt messages using Python & Streamlit. Perfect for sending digital Eidi to friends & family! 🎁✨

🚀 Features

Enter your name & friend’s name

Generate beautiful Eid card with:

🌙 Animated moon

✨ Flying stars & confetti

💌 Random Eid Mubarak messages & duas

Screenshot & share instantly on WhatsApp, Instagram, or Messenger

💻 Installation

Clone this repo:

git clone https://github.com/YourUsername/eid-card-generator.git

Install dependencies:

pip install streamlit

Run the app:

streamlit run app.py
🎬 How it Works

Open the app.

Enter your name and friend’s name.

Click Send Eid Card 💌.

Watch animated stickers & Eid Mubarak message appear!

Take a screenshot & share the joy!
